ما هي PHP ولماذا تتعلمها؟

What is PHP & Why?
PHP: Hypertext Preprocessor لغة مفتوحة المصدر لبناء مواقع الويب الديناميكية. الكود يشتغل على السيرفر وينتج HTML للمتصفح. بُني بها WordPress (43% من الإنترنت) وFacebook وWikipedia. أنا أستخدمها يومياً مع Laravel.

لماذا PHP؟

5 أسباب:
① سهلة للمبتدئين
② سوق عمل ضخم — Laravel مطلوب
③ تكامل ممتاز مع MySQL
④ Frameworks: Laravel، Symfony
⑤ Hosting رخيص في كل مكان
<?php
  $name = "Yahya";
  echo "مرحباً، $name!";
?>
مرحباً، Yahya!

كيف تعمل PHP؟

المستخدم يطلب الصفحة ← السيرفر يشغّل PHP ← ينتج HTML ← يُرسَل للمتصفح. المتصفح لا يرى كود PHP أبداً.
<?php
$hour = (int)date("H");
$greet = match(true) {
  $hour < 12 => "صباح الخير",
  $hour < 18 => "مساء الخير",
  default    => "مساء النور",
};
echo "<h1>$greet!</h1>";
?>

المتغيرات وأنواع البيانات

Variables & Data Types
المتغير صندوق يحفظ قيمة. في PHP كل متغير يبدأ بـ $. PHP لغة Loosely Typed — تحدد النوع تلقائياً. PHP 7+ تتيح Type Hints للأمان.

أنواع البيانات الـ 8

// string — نص
$name  = "Yahya Al-sulami";

// int — رقم صحيح
$age   = 25;
$big   = 1_000_000;  // PHP 7.4+

// float — رقم عشري
$price = 99.99;

// bool — صح أو غلط
$login = true;

// null — لا قيمة
$data  = null;

// array — مصفوفة
$tags  = ["php", "laravel"];

// object — كائن
$dt    = new DateTime();

// resource — ملف/DB مفتوح
$fp    = fopen("f.txt", "r");

فحص وتحويل الأنواع

$val = "42";

gettype($val);        // "string"
is_string($val);      // true
is_int($val);          // false
is_numeric($val);     // true
var_dump($val);        // string(2) "42"

// Casting
$i = (int)   $val;   // 42
$f = (float) $val;   // 42.0
$b = (bool)  $val;   // true
$s = (string)123;    // "123"

intval("15px");    // 15
floatval("3.5m");  // 3.5

الثوابت Constants

الثابت قيمته لا تتغير — يُكتب بـ CAPITALS.
define("SITE_NAME",  "Yahya.DEV");
define("MAX_UPLOAD", 5242880);
const API_VER = "v2.5";

echo SITE_NAME;  // Yahya.DEV
echo API_VER;    // v2.5

// ثوابت PHP المدمجة
echo PHP_VERSION; // 8.3.x
echo PHP_INT_MAX; // 9223372036854775807
echo PHP_EOL;     // سطر جديد
echo __FILE__;    // مسار الملف
echo __DIR__;     // مجلد الملف

نطاق المتغيرات Scope

المتغير الـ global غير مرئي داخل الدوال تلقائياً — يجب تمريره كـ parameter.
$site = "yahya.dev";  // global

function bad() {
  echo $site; // غير موجود!
}

// الافضل: parameter
function show(string $url): void {
  echo $url;
}
show($site);

// static — يتذكر قيمته
function counter(): int {
  static $n = 0;
  return ++$n;
}
echo counter(); // 1
echo counter(); // 2

العوامل والمقارنة

Operators & Comparison
أهم فرق في PHP: == (قيمة فقط) مقابل === (قيمة + نوع) — مصدر كثير من الأخطاء. دائماً استخدم ===.

حسابية + اختصارات

$a = 15; $b = 4;
echo $a + $b;   // 19 جمع
echo $a - $b;   // 11 طرح
echo $a * $b;   // 60 ضرب
echo $a / $b;   // 3.75 قسمة
echo $a % $b;   // 3 باقي
echo $a ** $b;  // 50625 اس

$x  = 10;
$x += 5;  // 15
$x -= 3;  // 12
$x *= 2;  // 24
$x /= 6;  // 4
$x++;     // post++
++$x;     // ++pre

المقارنة == vs ===

// == قيمة فقط — خطير!
5  ==  "5"   // true  - خطر
0  ==  false  // true  - خطر
0  ==  "foo"  // true  - خطر!

// === قيمة + نوع - صحيح
5  === "5"   // false - int vs string
5  === 5    // true  - صحيح
0  === false  // false - int vs bool

// Spaceship (PHP 7+)
1 <=> 2      // -1 اقل
2 <=> 2      // 0  متساوٍ
3 <=> 2      // 1  اكبر

منطقية + نصوص

true  && true   // true  AND
true  || false  // true  OR
!true            // false NOT

// دمج نصوص بـ .
$fn   = "Yahya";
$full = $fn . " Alsulami";
echo $full; // Yahya Alsulami

$t  = "Yahya.DEV";
$t .= " Full-Stack";
echo $t;

Null Coalescing و Ternary

// Ternary ?:
$s = ($age >= 18) ? "بالغ" : "قاصر";

// ?? Null Coalescing (PHP 7+)
$name = $_GET['name'] ?? "زائر";
$port = $cfg['port']  ?? 3306;

// ??= تعيين لو null (PHP 7.4+)
$cache ??= "default";

// سلسلة
$v = $a ?? $b ?? "fallback";

// ?: Elvis
$role = $user->role ?: "guest";

الشروط

Conditions
الشروط تتحكم في مسار التنفيذ. PHP عندها if / switch / match — كل واحد لحالته.

if / elseif / else

$score = 73;
if ($score >= 90) {
    echo "A — ممتاز";
} elseif ($score >= 80) {
    echo "B — جيد جداً";
} elseif ($score >= 65) {
        echo "C — جيد";  // هذا ينفذ لان 73 >= 65
} else {
    echo "F — راسب";
}

// مثال صلاحيات
$role = "editor";
if ($role === "admin") {
    echo "وصول كامل";
} elseif (in_array($role, ["editor", "author"])) {
    echo "وصول للكتابة";
} else {
    echo "قراءة فقط";
}

switch

break ضروري — بدونه يكمل للـ case التالية!
$method = "POST";
switch ($method) {
    case "GET":
        echo "جلب بيانات";
        break;
    case "POST":
                echo "ارسال";
        break;
    case "PUT":
    case "PATCH":
        echo "تحديث";
        break;
    default:
        echo "غير معروف";
}

match — PHP 8

يستخدم === تلقائياً، لا يحتاج break، يرجع قيمة مباشرة.
$code = 404;
$msg = match($code) {
    200, 201 => "نجاح",
    400       => "طلب خاطئ",
        404       => "غير موجود",
    500       => "خطأ سيرفر",
    default   => "مجهول",
};
echo $msg; // غير موجود

// match مع شرط بولياني
$disc = match(true) {
    $qty >= 100 => 0.20,
    $qty >= 50  => 0.10,
    default     => 0.00,
};

الحلقات

Loops
for لعدد محدد، while لشرط، foreach للمصفوفات — الاكثر استخداماً.

for

for ($i=1; $i<=5; $i++) {
    echo $i . " ";
}
// 1 2 3 4 5

// عكسي
for ($i=5; $i>=1; $i--) {
    echo $i . " ";
}
// 5 4 3 2 1

// متداخلتان
for ($r=1; $r<=3; $r++) {
    for ($c=1; $c<=3; $c++) {
        echo $r*$c . "\t";
    }
    echo "\n";
}

while و do-while

// while — يفحص اولاً
$p = 1;
while ($p <= 5) {
    echo "صفحة " . $p++ . "\n";
}

// do-while — ينفذ مرة اولاً
$tries = 0;
do {
    echo "محاولة اتصال\n";
    $ok = rand(0,1);
    $tries++;
} while (!$ok && $tries<3);

echo $ok ? "متصل" : "فشل";

foreach — الاكثر استخداماً

$skills = ["PHP","Laravel","MySQL"];

// قيمة فقط
foreach ($skills as $s) {
    echo "- $s\n";
}

// مع index
foreach ($skills as $i => $s) {
    echo ($i+1).". $s\n";
}

// Associative
$me = ["name"=>"Yahya","role"=>"Dev"];
foreach ($me as $k => $v) {
    echo "$k: $v\n";
}

break و continue

// continue — تخطي
for ($i=1; $i<=10; $i++) {
    if ($i%2===0) continue;
    echo $i." ";
}
// 1 3 5 7 9

// break — إيقاف
foreach ($users as $u) {
    if ($u["role"]==="admin") {
        $admin = $u; break;
    }
}

// break 2 — خروج من حلقتين
foreach ($rows as $row) {
    foreach ($row as $cell) {
        if ($cell===null) break 2;
    }
}

الدوال

Functions
الدالة كتلة كود معاد الاستخدام. مبدأ DRY: لا تكرر الكود — اجعله دالة. اللبنة الاساسية لأي كود نظيف.

تعريف واستدعاء

function formatPrice($p, $c="ج.م") {
    return number_format($p,2)." ".$c;
}
echo formatPrice(15000);
// 15,000.00 ج.م
echo formatPrice(250, "USD");
// 250.00 USD

// إرجاع متعدد
function minMax(array $nums): array {
    return [min($nums), max($nums)];
}
[$min, $max] = minMax([5,2,9,1]);
echo "Min:$min Max:$max"; // Min:1 Max:9

Type Hints (PHP 7+/8+)

تحديد انواع المدخلات والمخرجات — يمنع الاخطاء. Best Practice دائماً.
function add(int $a, int $b): int {
    return $a + $b;
}

// Nullable — قد يكون null
function find(int $id): ?string {
    return $id>0 ? "User#".$id : null;
}

// void — لا ترجع شيئاً
function log(string $msg): void {
    error_log(date("Y-m-d").": ".$msg);
}

// PHP 8: Union Types
function process(int|string $v): string {
    return (string)$v;
}

Arrow Functions و Closures

// Closure — دالة مجهولة
$double = function(int $n): int {
    return $n * 2;
};
echo $double(7); // 14

// use — جلب متغير خارجي
$tax  = 0.14;
$calc = function(float $p) use($tax): float {
    return $p * (1 + $tax);
};
echo $calc(100); // 114.0

// Arrow fn (PHP 7.4+) — ترث تلقائياً
$triple = fn(int $n) => $n * 3;
echo $triple(5); // 15

$addVAT = fn($p) => $p + $p*$tax;
echo $addVAT(200); // 228

Variadic و Spread

// ... يجمع كل المعاملات
function sumAll(int ...$nums): int {
    return array_sum($nums);
}
echo sumAll(1,2,3,4,5); // 15

// Spread — فرد مصفوفة
$nums = [10,20,30];
echo sumAll(...$nums); // 60

// First-Class Callables (PHP 8.1)
$words = ["php","laravel"];
$upper = array_map(strtoupper(...), $words);
// ["PHP","LARAVEL"]

المصفوفات

Arrays
المصفوفة تخزن مجموعة قيم في متغير واحد. PHP عندها 3 انواع: مرقمة وAssociative ومتعددة الابعاد. map/filter/reduce ستغيّر اسلوبك البرمجي.

إنشاء وعمليات

// Indexed — مرقم من 0
$stack = ["PHP","Laravel","MySQL"];
echo $stack[0]; // PHP

// Associative — مفتاح:قيمة
$proj = [
    "name"  => "Vortex Trader",
    "stack" => "Laravel",
    "live"  => true,
];
echo $proj["name"]; // Vortex Trader

// إضافة
$stack[] = "Redis";
array_push($stack, "Vue");
array_unshift($stack, "Nginx");

// حذف
array_pop($stack);
array_shift($stack);
unset($stack[1]);
echo count($stack);

map / filter / reduce

ثلاث دوال قوية لمعالجة البيانات بدون حلقات صريحة.
$prices = [100,250,80,400];

// map — دالة على كل عنصر
$taxed = array_map(
    fn($p) => round($p*1.14,2),
    $prices
);
// [114, 285, 91.2, 456]

// filter — فلتر بشرط
$exp = array_filter(
    $prices, fn($p) => $p>150
);
// [250, 400]

// reduce — قيمة واحدة
$total = array_reduce(
    $prices,
    fn($c,$p) => $c+$p, 0
);
// 830

الفورمز والجلسة

Forms, Sessions & Superglobals
Superglobals متغيرات PHP متاحة في كل مكان. القاعدة الذهبية: لا تثق ابداً في اي إدخال من المستخدم — دائماً تحقق ونظّف.

GET و POST

// GET — بيانات من URL
// URL: site.com?q=php&page=2
$q    = $_GET['q']    ?? '';
$page = (int)($_GET['page'] ?? 1);

// POST — معالجة فورم
if ($_SERVER['REQUEST_METHOD']==='POST') {
    $name  = trim($_POST['name']  ?? '');
    $email = trim($_POST['email'] ?? '');

    $errors = [];
    if (empty($name))
        $errors[] = "الاسم مطلوب";
    if (!filter_var($email,FILTER_VALIDATE_EMAIL))
        $errors[] = "إيميل خاطئ";

    if (empty($errors))
        echo "شكراً ".htmlspecialchars($name);
}

// SESSION — اول سطر في الصفحة

البرمجة كائنية التوجه

OOP
OOP طريقة تنظيم الكود. Laravel كله مبني عليها — لا يمكن استخدام اي Framework حديث بدون فهمها.

Class كامل — Constructor Promotion + Method Chaining

class Project {
    private       string $status = "draft";
    protected     array  $stack  = [];
    public static int    $count  = 0;

    // Constructor Promotion (PHP 8)
    public function __construct(
        public readonly string $id     = '',
        public          string $name   = '',
        public          string $client = "Personal",
    ) {
        self::$count++;
    }

    public function addTech(string ...$t): static {
        array_push($this->stack, ...$t);
        return $this;  // Method Chaining
    }
    public function launch(): void { $this->status = "live"; }
    public function getStack(): array { return $this->stack; }
    public static function getCount(): int { return self::$count; }
    public function __toString(): string {
        return "{$this->name} [{$this->status}]";
    }
}

قواعد البيانات — PDO

Database — PDO & MySQL
PDO الطريقة الصحيحة والآمنة للتواصل مع قواعد البيانات. لا تستخدم mysql_* ابداً. PDO تحمي من SQL Injection عبر Prepared Statements.

اتصال + CRUD كامل

// الاتصال
try {
    $pdo = new PDO(
        "mysql:host=localhost;dbname=mydb;charset=utf8mb4",
        "root", "",
        [
            PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
            PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
        ]
    );
} catch (PDOException $e) {
    die("فشل: ".$e->getMessage());
}

// SELECT — Prepared Statement (آمن)
$stmt = $pdo->prepare("SELECT * FROM users WHERE role=? AND active=?");
$stmt->execute(["admin", 1]);
$admins = $stmt->fetchAll();

// Named Parameters — اوضح
$stmt = $pdo->prepare("SELECT * FROM users WHERE id=:id");
$stmt->execute(["id"=>42]);
$user = $stmt->fetch();

// INSERT
$stmt = $pdo->prepare("INSERT INTO users(name,email,password) VALUES(:n,:e,:p)");
$stmt->execute([
    "n"=>"Yahya", "e"=>"y@dev.com",
    "p"=>password_hash("secret",PASSWORD_DEFAULT)
]);
$newId = $pdo->lastInsertId();

// UPDATE
$stmt = $pdo->prepare("UPDATE users SET role=? WHERE id=?");
$stmt->execute(["editor", 42]);
echo $stmt->rowCount(); // صفوف متاثرة

// DELETE
$pdo->prepare("DELETE FROM users WHERE id=?")->execute([42]);

// Transaction — الكل او لا شيء
try {
    $pdo->beginTransaction();
    $pdo->exec("UPDATE accounts SET balance=balance-500 WHERE id=1");
    $pdo->exec("UPDATE accounts SET balance=balance+500 WHERE id=2");
    $pdo->commit();
} catch (Exception $e) {
    $pdo->rollBack(); throw $e;
}

الامان

Security
الامان ليس اختيارياً. اكبر ثغرات PHP: XSS وSQL Injection وكلمات مرور غير مشفرة. الحل: htmlspecialchars + Prepared Statements + password_hash.

الحماية من اشهر الثغرات

// XSS Protection
$in = '<script>alert("hack")</script>';
// لا تستخدم مباشرة — استخدم:
echo htmlspecialchars($in, ENT_QUOTES, "UTF-8");

// دالة مساعدة
function e(string $s): string {
    return htmlspecialchars($s, ENT_QUOTES, "UTF-8");
}

// SQL Injection — Prepared Statement
// خطر:
$id = $_GET["id"];
$sql = "SELECT * FROM users WHERE id = $id"; // id = "1 OR 1=1"!
// آمن:
$stmt = $pdo->prepare("SELECT * FROM users WHERE id=?");
$stmt->execute([(int)$_GET["id"]]);

// Password Hashing
$pass = "mySecret123";
$hash = password_hash($pass, PASSWORD_DEFAULT);
if (password_verify($pass, $hash)) {
    echo "كلمة المرور صحيحة";
}

// CSRF Token
session_start();
$_SESSION["csrf"] ??= bin2hex(random_bytes(32));

    die("CSRF detected!");
}

class NotFoundException     extends RuntimeException {}
class UnauthorizedException extends RuntimeException {}

function getPost(int $id, int $uid): array {
    $post = fetchFromDB($id);
    if (!$post)
        throw new NotFoundException("المقال #$id غير موجود", 404);
    if ($post["user_id"] !== $uid)
        throw new UnauthorizedException("ليس لديك صلاحية", 403);
    return $post;
}

try {
    $post = getPost(42, 1);
    echo $post["title"];

    http_response_code(404);
    echo $e->getMessage();
} catch (UnauthorizedException $e) {
    http_response_code(403);
    echo $e->getMessage();
} catch (Throwable $e) {
    http_response_code(500);
    error_log($e->getTraceAsString());
    echo "خطأ في السيرفر";
