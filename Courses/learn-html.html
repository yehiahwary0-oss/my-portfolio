ما هي HTML؟

أساس كل صفحة ويب على الإنترنت

HTML تعني HyperText Markup Language — وهي ليست لغة برمجة، بل لغة ترميز تُخبر المتصفح بكيفية عرض المحتوى.

كل موقع ويب تراه على الإنترنت مبني في أساسه على HTML. إنها الهيكل العظمي للصفحة، مثل الهيكل الخرساني للمبنى — CSS هي الديكور، JavaScript هي الحياة.

HTML5 هو أحدث إصدار رسمي (2014 معتمد من W3C) وهو المعيار المستخدم اليوم بشكل كامل في جميع المتصفحات.

ℹ️
ملاحظة مهمة

HTML ليست لغة برمجة — لا تحتوي على متغيرات أو شروط أو حلقات. هي فقط تحدد هيكل المحتوى.

// STRUCTURE

البنية الأساسية لأي صفحة HTML

هيكل كل صفحة ويب بدون استثناء

كل صفحة HTML تبدأ بهذه البنية الجوهرية. احفظها جيداً — كل ما ستكتبه في حياتك كمطور يعتمد عليها.

index.html 
<!-- إعلان نوع المستند - دائماً في أول السطر -->
<!DOCTYPE html>

<!-- العنصر الجذر للصفحة - lang تحدد لغة الصفحة -->
<html lang="ar" dir="rtl">

<head>
  <!-- ترميز النصوص - يجب أن يكون أول شيء -->
  <meta charset="UTF-8" />

  <!-- التوافق مع الشاشات المختلفة -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />

  <!-- عنوان الصفحة يظهر في تبويب المتصفح -->
  <title>اسم صفحتي | Yahya.DEV</title>
</head>

<body>
  <!-- كل المحتوى الظاهر للمستخدم يأتي هنا -->
  <h1>مرحباً بالعالم!</h1>
</body>

</html>

// LINKS

الروابط <a>

العنصر الأكثر استخداماً في الويب

عنصر <a> (Anchor) هو جوهر الويب — إنه ما يربط الصفحات ببعضها. بدونه لا يوجد إنترنت بالمعنى الحقيقي.

links.html 
<!-- رابط خارجي - يفتح في تبويب جديد -->
<a href="https://yahya.dev"
   target="_blank"
   rel="noopener noreferrer">
  زيارة موقعي
</a>

<!-- رابط داخلي (نفس الموقع) -->
<a href="/about.html">عن الموقع</a>

<!-- رابط للانتقال لقسم في نفس الصفحة -->
<a href="#contact">اذهب للتواصل</a>

<!-- رابط بريد إلكتروني -->
<a href="mailto:yehiahwary515@gmail.com">
  راسلني
</a>

<!-- رابط هاتف -->
<a href="tel:+966XXXXXXXXX">اتصل بي</a>

<!-- رابط تنزيل ملف -->
<a href="/cv.pdf" download="yahya-cv">
  تنزيل السيرة الذاتية
</a>
الخاصيةالقيمالوظيفة
hrefURL / #id / mailto: / tel:وجهة الرابط
target_blank / _self / _parentأين يفتح الرابط
relnoopener / noreferrer / nofollowالعلاقة مع الصفحة المُشار إليها
downloadاسم الملفتنزيل بدل الفتح
titleنصتلميح يظهر عند التمرير
⚠️
أمان مهم

دائماً أضف rel="noopener noreferrer" مع target="_blank" — هذا يمنع هجوم Tab-Napping الذي يسمح للصفحة المفتوحة بالتحكم في صفحتك.

البنية الدلالية — Semantic HTML5

الفرق بين مطور عادي ومطور محترف

HTML الدلالي يعني استخدام العنصر الصحيح للمحتوى الصحيح. بدلاً من <div> لكل شيء، HTML5 قدم عناصر تصف معنى المحتوى.

هذا يؤثر على: SEO، إمكانية الوصول (Accessibility)، وسهولة قراءة الكود.

semantic-layout.html 
<body>

  <!-- رأس الموقع -->
  <header>
    <nav>
      <ul>
        <li><a href="/">الرئيسية</a></li>
      </ul>
    </nav>
  </header>

  <!-- المحتوى الرئيسي للصفحة -->
  <main>

    <!-- قسم من الصفحة له عنوان -->
    <section>
      <h2>أحدث المقالات</h2>

      <!-- محتوى مستقل قابل للنشر منفرداً -->
      <article>
        <header>
          <h3>تعلم HTML5</h3>
          <time datetime="2026-01-01">1 يناير 2026</time>
        </header>
        <p>محتوى المقال...</p>
        <footer>
          <address>بقلم: Yahya Al-sulami</address>
        </footer>
      </article>

    </section>

    <!-- محتوى جانبي (Sidebar) -->
    <aside>
      <h3>مواضيع ذات صلة</h3>
    </aside>

  </main>

  <!-- تذييل الموقع -->
  <footer>
    <p>© 2026 Yahya.DEV</p>
  </footer>

</body>

الـ Meta Tags وتحسين SEO

كيف تجعل Google تحب صفحتك

الـ Meta Tags لا تظهر للمستخدم لكنها تتحدث مع Google ومحركات البحث ومع منصات التواصل الاجتماعي عند مشاركة رابطك.

seo-meta.html 
<head>

<!-- تخزين بيانات مخصصة في HTML لاستخدامها بـ JS -->

  data-user-id="42"
  data-action="delete"

>
  حذف المستخدم

/* قراءتها في JavaScript */
const btn = document.querySelector('button');
